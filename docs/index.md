## React Grid Table 组件

rc-grid-table 是一个面向复杂数据表格场景的 React Table 组件，使用 CSS Grid 完成表格布局，提供固定列、虚拟列表、列拖拽排序、拖拽调整列宽、列显隐控制、树形数据、可编辑单元格和高级筛选行等能力。

## 适用场景

- 后台管理系统中的复杂数据表格
- 需要固定列、表头分组或列合并的业务表格
- 需要虚拟滚动、行拖拽排序、树形数据和可编辑单元格的数据列表
