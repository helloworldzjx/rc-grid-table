# 快速开始

## 安装

```bash
npm install rc-grid-table
```
or
```bash
yarn add rc-grid-table
```

## 使用
```js
import { Table } from "rc-grid-table"

<Table 
  columns={[]} 
  dataSource={[]} 
  resizableColumns 
  sortableColumns 
/>
```
