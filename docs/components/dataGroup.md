数据分组可以按树形数据渲染：父级记录承载聚合值，子级记录承载明细值。

这个示例在普通模式和数据分组模式之间切换时保持相同的列 `key`、顺序和层级，只切换 `dataSource`、标题文本和父级行渲染逻辑。分组模式下，父级行的非统计列渲染为空，统计列显示聚合值。

示例中分组后的父级行会在 body 滚动区域内吸顶，便于查看当前分组的聚合值。展开分组后，当前分组父行的 sticky cell 会使用阴影模拟显示底部 border；收起状态只保留吸顶位置。

`Group` 列使用受控排序，默认升序，排序方向只在升序和降序之间切换：

- 普通模式下，排序作用在平铺的明细数据上，按 `group` 字段排序。
- 数据分组模式下，排序作用在已经构造好的父级分组数据上，按每个分组的明细数量排序；数量相同再按 `group` 字段排序，明细数据仍保留在各自分组的 `children` 中。
